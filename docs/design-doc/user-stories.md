# User Stories

## As a User I want to be able to split an expense between people in a fair manner so that each person knows what they owe

 ### As a visitor (not signed in user)
 -  I want to quickly and efficiently create an expense.
 -  I want to add people to an expense so it can be shared evenly
 -  I want to generate an expense / summary result so that i can know who has paid for the expoense and how much each person ows to the payer.
 -  I want to see a summary of most recent expenses so i can have an overview snapshot when I open the app.

 ### As a logged in user
 -  I want to save an expense so that it can be accessed from any browser.
 -  I want to create an account so that my expenses and people can be stored
